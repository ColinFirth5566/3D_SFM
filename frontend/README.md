# Frontend

Next.js web application for 3D reconstruction.

## Setup

Install dependencies:
```bash
npm install
```

## Development

Run development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

Build for production:
```bash
npm run build
npm start
```

## Features

- Drag-and-drop image upload
- Real-time reconstruction progress tracking
- Interactive 3D model viewer with Three.js
- Download reconstructed models as GLTF files

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Three.js with React Three Fiber
- React Dropzone
